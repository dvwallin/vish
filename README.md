# vish
A shell coded in [V](https://vlang.io). Work in progress.

## Install and use
either
```
git clone https://github.com/dvwallin/vish.git
cd vish
v .
./vish
```

or
```
git clone https://github.com/dvwallin/vish.git
cd vish
v run vish.v
```
