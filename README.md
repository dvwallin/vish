# vlsh
A shell coded in [V](https://vlang.io). Work in progress.

## Install and use
either
```
git clone https://github.com/dvwallin/vlsh.git
cd vlsh
v .
./vlsh
```

or
```
git clone https://github.com/dvwallin/vlsh.git
cd vlsh
v run vlsh.v
```
